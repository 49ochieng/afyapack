# Starter: Empty Project

You're starting from scratch. Follow Lessons 00-02 to set up your environment.

## What You Need

1. **VS Code** with GitHub Copilot extension
2. **Node.js** v18 or higher
3. **Foundry Local** - Install from VS Code AI Toolkit

## First Steps

1. Complete Lesson 00 (Introduction)
2. Complete Lesson 01 (Setup)
3. Complete Lesson 02 (Foundry Local)
4. Then start Lesson 03 to scaffold your project

## Verify Your Setup

Before proceeding, confirm:

```bash
# Node is installed
node --version  # Should be v18+

# npm is available
npm --version

# Foundry Local is running
curl http://127.0.0.1:54346/v1/models
```

Then return to the labs folder and start Lesson 03.
