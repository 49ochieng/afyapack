# Checkpoints

This folder contains reference snapshots showing what the code should look like after completing each lesson.

## Usage

If participants fall behind or get stuck:

1. Compare their code with the checkpoint
2. Help identify differences
3. Use checkpoint as reference, not copy-paste

## Where to Find Code

The complete final code is in the main project folders:
- `api/` - Complete backend
- `web/` - Complete frontend

Each lesson README contains the expected code for that lesson.

## Note for Instructors

- Checkpoints are not meant for copying
- Walk through the code to explain what's happening
- Encourage participants to type the code themselves
- Use Copilot prompts from lessons to regenerate if needed
