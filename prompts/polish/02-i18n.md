# Prompt: Internationalization (i18n)

## Description
Implement multilingual support in a React application with language switching, translation management, and locale-aware formatting.

---

## The Prompt

```
Create an i18n system for React that:
1. Supports multiple languages (English, Swahili)
2. Uses React Context for language state
3. Stores translations in organized JSON/JS files
4. Provides a `t()` function for translations
5. Handles pluralization and interpolation
6. Persists language preference
7. Includes a language switcher component
8. Supports RTL languages (future-proof)

Design for a healthcare application with medical terminology.
```

### lib/i18n/context.js
```jsx
'use client';

import { createContext, useContext, useState, useCallback, useEffect, useMemo } from 'react';
import en from './translations/en';
import sw from './translations/sw';

const translations = { en, sw };

const I18nContext = createContext(null);

export function I18nProvider({ children, defaultLang = 'en' }) {
  const [lang, setLang] = useState(defaultLang);
  
  // Load from localStorage
  useEffect(() => {
    const saved = localStorage.getItem('afyapack-lang');
    if (saved && translations[saved]) {
      setLang(saved);
    }
  }, []);
  
  // Save changes
  const changeLang = useCallback((newLang) => {
    if (translations[newLang]) {
      setLang(newLang);
      localStorage.setItem('afyapack-lang', newLang);
      // Update html lang attribute
      document.documentElement.lang = newLang;
    }
  }, []);
  
  // Translation function with interpolation
  const t = useCallback((key, variables = {}) => {
    let text = translations[lang]?.[key] || translations.en?.[key] || key;
    
    // Handle interpolation: t('greeting', { name: 'John' }) => "Hello, John"
    Object.entries(variables).forEach(([varKey, value]) => {
      text = text.replace(new RegExp(`{{${varKey}}}`, 'g'), value);
    });
    
    return text;
  }, [lang]);
  
  // Pluralization helper
  const plural = useCallback((key, count) => {
    const pluralKey = count === 1 ? key : `${key}_plural`;
    return t(pluralKey, { count });
  }, [t]);
  
  const value = useMemo(() => ({
    lang,
    setLang: changeLang,
    t,
    plural,
    isSwahili: lang === 'sw',
    isEnglish: lang === 'en',
    languages: [
      { code: 'en', name: 'English', nativeName: 'English' },
      { code: 'sw', name: 'Swahili', nativeName: 'Kiswahili' }
    ]
  }), [lang, changeLang, t, plural]);
  
  return (
    <I18nContext.Provider value={value}>
      {children}
    </I18nContext.Provider>
  );
}

export function useI18n() {
  const context = useContext(I18nContext);
  if (!context) {
    throw new Error('useI18n must be used within I18nProvider');
  }
  return context;
}
```

### components/LanguageSwitcher.jsx
```jsx
'use client';

import { useI18n } from '@/lib/i18n/context';
import { Globe } from 'lucide-react';

export default function LanguageSwitcher({ variant = 'dropdown' }) {
  const { lang, setLang, languages } = useI18n();
  
  if (variant === 'toggle') {
    return (
      <button
        onClick={() => setLang(lang === 'en' ? 'sw' : 'en')}
        className="flex items-center gap-2 px-3 py-2 rounded-lg hover:bg-gray-100 transition-colors"
      >
        <Globe className="w-4 h-4" />
        <span className="font-medium">
          {lang === 'en' ? 'SW' : 'EN'}
        </span>
      </button>
    );
  }
  
  return (
    <div className="relative">
      <select
        value={lang}
        onChange={(e) => setLang(e.target.value)}
        className="appearance-none bg-transparent pl-8 pr-4 py-2 rounded-lg border border-gray-200 hover:border-gray-300 focus:outline-none focus:ring-2 focus:ring-blue-500"
      >
        {languages.map(language => (
          <option key={language.code} value={language.code}>
            {language.nativeName}
          </option>
        ))}
      </select>
      <Globe className="absolute left-2 top-1/2 -translate-y-1/2 w-4 h-4 text-gray-500 pointer-events-none" />
    </div>
  );
}
```

## Usage Examples

```jsx
import { useI18n } from '@/lib/i18n/context';

function MyComponent() {
  const { t, lang, isSwahili } = useI18n();
  
  return (
    <div>
      <h1>{t('chat_welcome')}</h1>
      <p>{t('chat_welcome_subtitle')}</p>
      
      {/* With interpolation */}
      <p>{t('greeting', { name: 'John' })}</p>
      
      {/* Conditional content */}
      {isSwahili && <p>Karibu sana!</p>}
    </div>
  );
}
```

## Adding New Languages

1. Create translation file: `lib/i18n/translations/[code].js`
2. Add to imports in context.js
3. Add to `translations` object
4. Add to `languages` array
